# Social-Media-Engagement-Dashboard-Instagram-Facebook-Created-by-Soham-
🔹 Project Title

Social Media Engagement Dashboard (Instagram & Facebook)

🔹 Short Description

This dashboard provides a comprehensive analysis of social media performance across Instagram and Facebook. It tracks key engagement metrics like likes, comments, shares, reach, and impressions to help understand audience behavior and content effectiveness.

🔹 Process
Data collected and cleaned in Excel
Data imported into Power BI
Data transformation using Power Query
Created calculated measures (Clicks, Engagement, Reach, etc.)
Designed interactive visuals (Donut Chart, Bar Chart, KPI Cards)
Final dashboard built with proper layout and formatting
🔹 Tech Stack
Power BI
Microsoft Excel
Power Query
DAX (Data Analysis Expressions)
🔹 Data Source
Sample social media dataset (Instagram & Facebook posts data)
Fields include: Likes, Comments, Shares, Reach, Impressions, Platform, Post Type
🔹 Features & Highlights
📊 Interactive and user-friendly dashboard
📈 Platform-wise performance comparison
👍 Engagement analysis (Likes, Comments, Shares)
📅 Year-wise impressions tracking
🔍 Post-type performance insights (Reels, Images, Videos)
🎯 Clear visualization for decision-making
